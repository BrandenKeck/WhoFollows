About This Application:

This is a text-based application based on the film, It Follows. Here is how it works: a hypothetical monster begins "chasing" a random person from Twitter. Then, the monster can either be passed on to one of that person's followers or, if the monster catches them, it will revert back to its previous target. The monster's actions as well as a list of targets can be tracked above. This is a fully automated Twitter "bot" that runs using Node.js and various front-end technologies.
